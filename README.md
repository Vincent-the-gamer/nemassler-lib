# nemassler-lib

> [!NOTE]
> Work in progress.

# Usage

## Install dependency
```shell
npm i nemassler-lib
```

## Use in Node.js
```ts
import { ncm2mp3 } from "nemassler-lib"

const result = ncm2mp3("/xxx/ncm","/xxx/mp3")
```

# Dev
```shell
pnpm i && cargo check

pnpm run debug
```
